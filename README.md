# sturdy-engine

## Getting Started

This repository now uses Electron Forge to provide the Electron build pipeline for the desktop app.

### Prerequisites

- Node.js 16.4 or newer
- Git
- npm

### Install dependencies

```bash
cd desktop-app
npm install
```

### Start the app

```bash
cd desktop-app
npm start
```

### Run tests

```bash
cd desktop-app
npm run test
```

### Build the renderer

```bash
cd desktop-app
npm run build
```

### Package distributables

```bash
cd desktop-app
npm run package
npm run make
```

### Publish artifacts

```bash
cd desktop-app
npm run publish
```
